CCS : Cascading Style Sheet ou feuille de style en cascade

A quoi ça sert ?

Le CSS est un langage informatique utilisé sur l'internet pour mettre en forme les fichiers HTML ou XML. Ainsi, les feuilles de style, aussi appelées les fichiers CSS, comprennent du code qui permet de gérer le design d'une page en HTML.

Un peu d’histoire :

Contrairement aux logiciels, les spécification CSS ne sont pas développées par versions successibles, qui permettraient à un navigateur de se référer à une version en particulier. CSS est développé par « Niveau », ce qui contraint à chaque nouveau niveau à intégrer le précédent, et chaque implémentation à être compatible avec les précédentes : CSS1 est donc développé pour être un sous ensemble de CSS2, qui lui-même est développé pour être un sous ensemble de CSS3. Ceci explique en partie la lenteur de l’avancement normatif de CSS. Le langage CSS a été créée par la W3C en 1990 et totalement pris en charge par les navigateurs dans les années 2000. La première version vu le jour en mai 1995, pour développer les feuilles de styles, par la suite en décembre 1996 devient une « recommandation » du W3C sous le nom de CSS1. CSS1 ne décrit pas uniquement sa propre grammaire : ce premier niveau décrit en effet une grammaire destinée à permettre aux niveaux ultérieurs d'ajouter de nouvelles fonctionnalités. En 1996, internet explorer 3.0 est le premier navigateur commercial à implémenter partiellement CSS alors que celle-ci est encore en cours de formulation. En 1998, alors que s'achève la guerre des navigateurs opposant Netscape et Microsoft, Opera 3.5, dont Håkon Lie est entre-temps devenu le directeur technique, parvient à une implémentation plus complète de CSS1. En mai 1998, le second niveau de CSS étend considérablement les possibilités théoriques des feuilles de styles en cascade, avec en particulier environ 70 propriétés supplémentaires. Les retours d'implémentation de CSS2 conduisent le groupe de travail CSS du W3C à rédiger, à partir de 2001, une version révisée CSS 2.1 (« CSS Level 2, Revision 1 »). Le développement du troisième niveau des feuilles de styles en cascade commence dès 1999, parallèlement à celui de CSS 2.1. CSS3 devient « modulaire », afin de faciliter ses mises à jour.

L’avenir de CSS ?

On a l'impression qu'il y a eu une petite accélération au cours des cinq dernières années de nouvelles fonctionnalités qui ont fait leur chemin dans CSS et une approche beaucoup plus ouverte et collaborative du w3c avec de vrais spécialistes indépendants.

Le HTML

Le langage HTML (Hyper Texte Markup Langage) imaginé et créé à partir de 1989 par Tim Berners-Lee au sein du bâtiment 31 du CERN basé à Saint Genis-Pouilly dans le département de l’Ain est à la base de la création du web avec les technologies HTTP et URL. Ce langage permet comme son nom l’indique d’écrire de l’hypertexte, de mettre en forme et structurer une page web. Il permet aussi de créer des formulaires de saisie et d’inclure du contenu multimédia (images, vidéos, programmes informatiques). Aujourd’hui il est presque toujours couplé avec le CSS et complété ensuite par le langage de programmation JavaScript. Cependant le HTML c’est suffit à lui-même avant l’apparition du langage CSS et JS. Il a alors subit plusieurs évolutions depuis sa création. En effet, nous utilisons aujourd’hui le HTML5.

Chronologie :

  • -1989-1992 : Origine
  • -1993 HTML 1.0
  • -1995-1996 : HTML 2.0
  • -1997 : HTML 3.2 (html 3.0 fut un brouillon) et 4.0
  • -2000-2006 : XHTML
  • -De 2007 à nos jours : HTML 5 et abandon du XHTML 2
