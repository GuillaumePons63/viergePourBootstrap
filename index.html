Le WEB, définition :


Le web fait référence au World Wide Web (WWW), le système d’interconnexion d’une base de données ou serveur à un terminal client via le protocole http, hypertext transfer protocol. Web est un terme anglais voulant dire toile. L’interconnexion des différents terminaux informatique permettant la communication entre eux représente une toile informatique.

Réseau internet :


Pour que le principe du web puisse fonctionnement à l’échelle mondial, il a besoin d’une structure physique qu’on appelle le réseau internet. Le réseau internet est formé de câble à fibre optique de plusieurs milliers de km de long, immergés dans l’océan, permettant de raccorder un pays à un autre. Ainsi des informations stockées sur des serveurs dans un pays peuvent être requêter, afficher et lu par un client raccordé au réseau internet via un pc et un navigateur web. Le trafic internet via ces câbles sous-marins représente aujourd’hui 99% des échanges d’informations du monde. Il existe également une possibilité de transférer des données de manière sans fil grâce aux satellites. La transmission des données est alors effectuée par radiofréquence. Historiquement, le premier réseau mondial à câble sous-marins était celui du télégraphe.

Fonctionnement du Web :


Pour créer le Web nous avons besoin d’un client et d’un serveur. Le client est un appareils utilisateurs connectés à internet et le serveur est un ordinateur qui stock des pages web, sites, etc... Le client envoie une requête au serveur pour accéder à une page web ou autres. Une copie de la page web sera téléchargé par le serveur pour l’envoyer au client. A suivre …
