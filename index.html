Les extensions VS Code

1/ Live Server : Permet de lancer un mini serveur qui va automatiquement rafraichir le visuel de la page html et css à chaque enregistrement. Permet d’éviter de cliquer des centaines de fois par jour sur rafraichir.


2.1/ Prettier: Permet de formater le code et de l’indenter automatiquement.


2.2/ ESLint : Linter, minifie, corrige et indente le code.


3/ Live Share : Permet de partager à plusieurs un code et de travailler dessus en même temps.


4/ Color Highlight : Permet d’avoir un meilleur visuel des codes couleur utilisés dans le code.


5/ Auto rename Tag : Modifie automatiquement le nom de la balise sur laquelle le nom est modifié.
